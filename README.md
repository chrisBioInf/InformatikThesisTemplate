This is a LaTeX document class of the Bioinformatics institute to help you write a PhD thesis compliant with the rules of our faculty, i.e., the "Promotionsordnung" of the Faculty of Mathematics and Computer Science at Leipzig University. With some modifications, it may be useful for writing a Master's or a Bachelor's thesis, too.

Originally, it was written by Fabian Gärtner, a former PhD student of the Bioinf. He himself and others have successfully used it for their own theses. The current release of the template is available for download here.

Maintenance and further development is carried out by volunteering users. The development is managed in a Git repository phd-thesis-class hosted on our local Gitolite server. To check out the repository, set up your Gitolite access (cf. here) and run git clone gitolite:phd-thesis-class.

For more information, please refer to the extensive manual.pdf of the template.
